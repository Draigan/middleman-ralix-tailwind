# Middleman-Ralix template

Starter Kit to build static websites powered with:

- [Middleman](https://middlemanapp.com)
- [Webpack](https://github.com/webpack/webpack)
- [Ralix](https://github.com/ralixjs/ralix)

## Install

```
> bin/setup
```

## Run

```
> bin/start
```

## Build

```
> bin/build
```
